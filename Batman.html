Overview.

The card game was created to learn the Javascript programming language. The goal was to learn classes and object programming. In the game I also used a new library which is Tilt Js. Thanks to it I created a nice 3d effect. The game is inspired by my favorite character from the world DC Batman.

Chalange.

The biggest challenge was to understand the new Javascript themes, which gave me a lot of problems. Creating this game took a lot of self-denial and finding information to solve the problem. The Scrimb tutorial helped me a lot in writing

Results.

In the game, my main inspiration was the city of gotham city. That's why I wanted to keep the dark climate. In creating the cards, I used Tilt Js, thanks to which I obtained the 3d effect. The colors I used are mainly black and yellow. The font is borrowed from the Batman comic, I had a lot of fun writing this game.
